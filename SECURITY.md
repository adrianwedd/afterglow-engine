# Security Policy

This project does not process untrusted network data and is intended for local/offline use. If you discover a security issue (e.g., path traversal, unsafe file handling), please open an issue with details or contact the maintainer directly.

## Supported Versions
- Current `master` branch.

## Reporting
- Include environment details (OS, Python version) and the exact command used.
- Do not include sensitive or licensed audio in reports; describe the file characteristics instead.
