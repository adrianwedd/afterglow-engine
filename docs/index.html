The Philosophy

This tool does not compose.
It does not invent.

It is kept in the back of the studio. Its purpose is to ingest existing audio archives and unweave them. It separates pigment from gesture. It mines for surfaces, atmosphere, and dust.

"We are distilling memories, not processing signals."

The Process

From archive to texture

Five stages of sonic distillation. Each pass extracts a different layer of memory.

🔍

Pre-Analysis

Surveying the dig site.

Scans for stability — RMS energy, onset density, spectral flatness. Only worthy regions pass the gate.

◼

Pad Mining

Extracting the surfaces.

Finds sustained, stable moments in full tracks. Extracts loopable segments with phase-aligned crossfades.

◎

Drone & Swell Generation

Stretching time into infinity.

Time-stretches and pitch-shifts tonal material into infinite drones and evolving one-shot swells. Warm, airy, dark variants.

☁

Granular Clouds

Dissolving memory into atmosphere.

Breaks audio into grains, filters for quality, reassembles into evolving cloud textures with pitch and time variation.

✧

Hiss & Dust

The residue of rhythm.

Extracts high-frequency content — the air, the hiss, the transient flickers. Creates tremolo loops and flicker bursts from the ghost of percussion.

Control

Tune every threshold

One YAML file. Every parameter documented, every threshold validated.

click to copy# config.yaml — the control surface

pad_miner:
  rms_threshold_db: -35        # minimum energy to extract
  max_onset_rate: 3.0          # lower = more sustained
  loop_crossfade_ms: 50        # seamless loop points

clouds:
  grain_length_ms: [40, 120]   # grain size range
  grain_count: 200              # density of the cloud
  pitch_range_semitones: [-2, 2] # subtle pitch scatter

curation:
  auto_delete_grade_f: true     # silence and clipping, gone
  grade_thresholds:
    a_min_rms: -25              # only the best survive
Usage

Run the machine

One command to mine. Individual stages when you want precision.

click to copy$ python make_textures.py --all           # full pipeline
$ python make_textures.py --mine-pads     # surfaces only
$ python make_textures.py --make-clouds   # atmosphere only
$ python make_textures.py --make-hiss     # dust only
